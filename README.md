# Proxy

Портативный прокси-клиент для Windows на базе ядра [sing-box](https://github.com/SagerNet/sing-box).
Работает **без установки**, перехватывает трафик и заворачивает его через прокси. GUI на
[Wails](https://wails.io) (Go + WebView2), ядро — как отдельный процесс, управляется через Clash API.

> Версия **1.0.0**

## Возможности

- **Протоколы:** VLESS (Reality / XTLS-Vision / ws / grpc / http / httpupgrade), VMess, Trojan,
  Shadowsocks, Hysteria2, TUIC, AnyTLS. Плюс **XHTTP** — при использовании альтернативного ядра
  (см. [«Два ядра»](#два-ядра)).
- **Два режима перехвата:**
  - *Системный прокси* — без прав администратора;
  - *TUN* — весь трафик системы (запрос UAC, `wintun.dll` в комплекте).
- **Профили и подписки:** ссылки `vless://…`, готовый JSON или подписки (sing-box JSON /
  Clash YAML / base64).
- **Маршрутизация:** «весь трафик через прокси» или сплит-туннель «РФ напрямую» (гео rule-set
  `.srs`), опциональный блок рекламы.
- **Живая статистика** (скорость ↑/↓, соединения), **пинг нод**, переключение и **авто-выбор**
  лучшей ноды (urltest) через Clash API.
- **Отделка:** трей со сворачиванием, автозапуск с Windows, аккуратная очистка системы при выходе
  и падении ядра (job object убивает ядро вместе с GUI).

## Скачать и запустить

Готовое приложение — в разделе [**Releases**](../../releases): распакуй
`Proxy-portable-<версия>-win64.zip` в любую папку (или на флэшку) и запусти `Proxy.exe`.
Установка не требуется, данные (профили/настройки) хранятся рядом в `data\`.

## Два ядра

В архиве два ядра в `assets\`:

| Файл | Ядро | Назначение |
|------|------|-----------|
| `sing-box.exe` | официальное (стабильное) | по умолчанию |
| `sing-box-xhttp.exe` | форк [sing-box-lx](https://github.com/Leadaxe/sing-box-lx) (`with_xhttp`) | для нод `type=xhttp` |

Переключение — в приложении: строка **«Ядро» → «Выбрать ядро…»**. Выбор запоминается и
переносится вместе с папкой (портативно, с любой буквы диска).

## Сборка из исходников

Нужны: [Go](https://go.dev) 1.23+, [Wails v2](https://wails.io/docs/gettingstarted/installation),
Node.js.

```sh
wails build          # соберёт build\bin\Proxy.exe
```

> Бинарники ядра/`wintun.dll`/гео-базы (`assets\*.exe`, `*.dll`, `*.db`, `*.srs`) в git не хранятся
> — возьми их из архива в [Releases](../../releases) и положи в `assets\` перед сборкой.

Тесты: `go test ./...`; интеграционные (нужен реальный `sing-box.exe`) — под тегом `coretest`
с переменными `PROXY_ASSETS` и `PROXY_TEST_LINK`.

## Технологии

Go · Wails v2 · Svelte + TypeScript · sing-box · Clash API

## Лицензии

GUI — авторский. Ядро sing-box и форк sing-box-lx распространяются под **GPLv3** и подключаются как
отдельные процессы; их исходники — по ссылкам выше. В комплект также входят базы geoip/geosite.
